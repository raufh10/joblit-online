# Joblit 🟢

**Joblit** is a specialized search engine designed to pave the way for careers in Data and AI. It uses semantic embedding to match user queries with specific roles like Data Analyst, Data Engineer, and AI Engineer.

## 🚀 Live Demo
You can view the live site here: [https://raufh10.github.io/joblit-online/](https://raufh10.github.io/joblit-online/)

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern Flexbox), Vanilla JavaScript (ES6+).
* **Backend:** FastAPI (Python) hosted on Railway.
* **Database:** Supabase.
* **AI:** Sentence-Transformers (`all-MiniLM-L6-v2`) for role matching.
